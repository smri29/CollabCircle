# CollabCircle

CollabCircle is being developed as a two-app workspace:

- `backend/`: NestJS API
- `frontend/`: Next.js web application

This repository is being built incrementally. The first iteration initializes the backend and sets the base repository structure for team collaboration.

## Current Status

- Root repository scaffolding is in place.
- `backend/` has been initialized with a NestJS starter project.
- `frontend/` is reserved for the upcoming Next.js setup.

## Project Structure

```text
CollabCircle/
|-- backend/
|-- frontend/
|-- .gitignore
`-- README.md
```

## Backend Commands

Run these commands from [`backend/`](/e:/myWebsites/CollabCircle/backend):

```bash
npm install
npm run start:dev
```

Other useful commands:

```bash
npm run build
npm run test
npm run test:e2e
```

## Collaboration Approach

The project will be developed in small iterations. Each completed step can be reviewed, committed, and pushed before the next one begins.
